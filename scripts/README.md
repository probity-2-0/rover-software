# Scripts Folder

This folder is for experimental and development scripts.

## Purpose
- Used for testing features before integrating into ROS2 packages
- Each team member works in their own script file
- Code here is not final production code

## Rules
- Each member creates their own file:
  scripts/<yourname>_color_detection.py

- Do NOT edit someone else's file
- Do NOT place files outside this folder

## Workflow
1. Create your own branch
2. Add your script here
3. Commit and push
4. Open a Pull Request to `dev`

## Note
After review, the best implementations will be merged into the main ROS2 vision package.
